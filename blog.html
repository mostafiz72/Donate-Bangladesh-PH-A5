What is the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each element of the document (like a paragraph, a header, or an image) is a node in the tree. This allows programming languages like JavaScript to interact with the document, modifying its content, structure, and style dynamically.

What is even delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM that allows you to handle events at a higher level in the DOM hierarchy, rather than attaching event listeners to individual elements. Instead of adding an event listener to every child element, you can add a single event listener to a common parent, and through event bubbling, capture events triggered by child elements.
